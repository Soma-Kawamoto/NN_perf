#### 3.1. `1. GPR Regression.py`

これは、B-Hヒステリシスループをガウス過程回帰（GPR）を用いてモデル化し、任意の磁束密度振幅におけるヒステリシスループを予測（回帰）するためのメインスクリプトです。

**主な機能:**

-   **モデル学習:**
    -   `1.Training Data Folder`から、指定された振幅の学習用ヒステリシスループデータ（ダウンサンプリング済み）を読み込みます。
    -   オプションで、`2.Normal Magnetization Curve Extraction Folder`で作成された正規磁化曲線のデータ（Akimaスプライン補間点）を学習に追加し、ループの先端部分の精度を向上させることができます。
    -   GPyライブラリを用いてGPRモデルを構築し、ハイパーパラメータの最適化（学習）を行います。
-   **回帰（予測）:**
    -   学習済みモデルを使い、指定された範囲の磁束密度振幅（`Bmreg_min`から`Bmreg_max`まで）に対応するヒステリシスループを予測します。
-   **精度評価と結果出力:**
    -   予測結果を正解データ（`7.reference data`）と比較し、RMSE（二乗平均平方根誤差）を計算してコンソールに表示します。
    -   学習データ、回帰結果と正解データの比較グラフを画面に表示します。
    -   詳細な結果（RMSEサマリ、各振幅での予測値と正解値の比較データ、比較グラフ）をExcelファイルとして`3.Answer`フォルダに出力します。

**設定と実行:**

-   **設定ファイル:**
    -   動作はすべて`config/1. GPR Regression.ini`ファイルで制御します。
    -   `PERFORM_TRAINING`: `True`にするとモデルの学習から実行します。`False`にすると、以前学習させた際に保存されたハイパーパラメータを読み込んで高速に回帰処理のみを実行します（設定が一致する場合）。
    -   `USE_AKIMA_DATA`: `True`にすると正規磁化曲線のデータを学習に加えます。
    -   その他、材料名、周波数、学習・回帰の振幅範囲などを指定できます。
-   **実行:**
    -   `src`フォルダにある`1. GPR Regression.py`を実行します。
