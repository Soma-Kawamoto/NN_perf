## 2. 動作説明

### 2.1. `1. NN.py`

これは、ディープニューラルネットワーク（DNN）を用いてB-Hヒステリシスループをモデル化し、任意の磁束密度振幅におけるループを予測するためのスクリプトです。

**主な機能:**

-   **モデル学習:**
    -   `1.Training Data Folder`から学習用ヒステリシスループデータを読み込みます。
    -   正規磁化曲線のデータ（Akimaスプライン補間点）を学習に追加し、ループ先端の精度向上を図ります。
    -   PyTorchライブラリを用いて全結合型ニューラルネットワークモデルを構築し、学習（重みの最適化）を行います。
-   **回帰（予測）:**
    -   学習済みモデルを使い、指定された範囲の磁束密度振幅に対応するヒステリシスループを予測します。
-   **精度評価と結果出力:**
    -   予測結果を正解データ（`7.reference data`）と比較し、RMSE（二乗平均平方根誤差）を計算してコンソールに表示します。
    -   学習データ、回帰結果と正解データの比較グラフを画面に表示し、`3.Answer/NN_regression_results`フォルダ内に画像ファイルとしても保存します。
    -   詳細な結果（RMSEサマリ、各振幅での予測値と正解値の比較データ、比較グラフ）をExcelファイルとして`3.Answer/NN_regression_results`フォルダに出力します。

**設定と実行:**

-   **設定ファイル:**
    -   **（注意）** このスクリプトは`config`フォルダの`.ini`ファイルを**使用しません**。
    -   設定はすべて`1. NN.py`スクリプトの先頭部分にある「ユーザー設定箇所」で直接変更します。
    -   `PERFORM_TRAINING`: `True`にするとモデルの学習から実行します。`False`にすると、以前学習させた際に保存されたモデルとスケーラーを読み込んで高速に回帰処理のみを実行します（設定が一致する場合）。
    -   その他、材料名、周波数、学習・回帰の振幅範囲、ニューラルネットワークの構造（隠れ層の数やノード数）、学習率などを指定できます。
-   **実行:**
    -   `src`フォルダにある`1. NN.py`を実行します。
