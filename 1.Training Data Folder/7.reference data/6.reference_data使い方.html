プログラム概要説明 Script Overview

📁 入力ファイル

本プログラムは、特定の材料・周波数・振幅におけるB-H波形データが記録されたExcelファイルを読み込みます。

フォルダ構成:

{BASE_PATH}/3.Fourier Transform Correction/{MATERIAL_NAME}/{FREQ}Hz/

ファイル名形式:

Bm{振幅}hys_{周波数}hz.xlsx (例: Bm1.2hys_50hz.xlsx)

ファイル内容:

2列の数値データ（1列目: H波形, 2列目: B波形）が1024行記録されています。

📄 出力ファイル

入力された波形データを補正・補間し、サマリーExcelファイルとして出力します。ファイルは`step_size`ごとに生成されます。

フォルダ構成:

{BASE_PATH}/7.reference data/

ファイル名形式:

summary_{MATERIAL_NAME}_{STEP_SIZE}.xlsx (例: summary_50A470_0.05.xlsx)

ファイル内容:
  • 周波数ごとに1つのシートが作成されます (例: `50Hz`シート)。
  • 各シートには、処理された複数のヒステリシスループデータが格納されます。
  • 列構成: B, H_ascending (B増加時), H_descending (B減少時)。
  • シートの先頭行には、処理した振幅の数、最大振幅、ステップサイズなどのメタデータが記録されます。

⚙️ 処理の概要

このプログラムは、以下のステップでデータの処理を実行します。

  1. ループ処理: 設定されたSTEP_SIZES_TO_PROCESSとFREQ_ARRAYに基づき、各周波数・各振幅の入力ファイルを順番に処理します。
  2. 波形補正: 読み込んだB-H波形データに対してフーリエ変換(FFT)を適用します。DCオフセット等の影響を除去するため、奇数次の高調波成分のみを抽出し、逆フーリエ変換(IFFT)によって波形を再構築します。
  3. 経路分割: 補正された1サイクルのループデータから、磁束密度Bが増加していく側のパス（ascending）と、減少していく側のパス（descending）を特定し、分割します。
  4. データ補間: 全てのループを共通の基準で比較できるよう、新しく等間隔なB軸を作成します。その後、分割した各パスのデータを用いて線形補間を行い、新しいB軸に対応するHの値を算出します。
  5. 集約と出力: ある一つの周波数に対する全ての振幅の補間済みループデータを集約し、対応するシートに出力します。この処理を全ての周波数で繰り返し、最終的に一つのサマリーファイルとして保存します。
