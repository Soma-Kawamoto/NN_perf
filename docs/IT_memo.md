# IT用語メモ

## 2025/12/20 ipconfigについて

### IPv4アドレス(IP: Internet protocol)
- PCの住所のこと。世界中でこのPCを特定するための番号(172.xx.xx.xx など)

### サブネットマスク
- 住所の範囲(グループの大きさ)を決めるもの。どこまでが同じネットワーク内の中まで、どこからが外の世界かを定義している
- (255. 255. 255. 0)の時、172. xx. xxまでが同じなら仲間、最後の数字は部屋番号のようなもの

### デフォルトゲートウェイ
- 外の世界への出口(ルーター)のこと → あs部ネットの外にアクセスしたいとき、ここにデータを投げれば外につないでくれるとう相談窓口のこと

### イーサネット アダプター イーサネット(Ethernet: エーテル+ネットワークが語源; ボブ・メトカーフが命名した。エーテルの様にデータが満ちて伝わるネットワークのこと)
- 優先LANケーブルを指すための接続口のこと。IPアドレスが表示されている場合、LANケーブルでインターネットにつながっているということ

### Wireless Lan adapter Wi-Fi
- Wi-Fiのこと。

### リンクローカルIPv6アドレス
- すぐ隣の機器とだけ話すための新しい形式の仮住所のこと。IPv4が世界中で足りなくなったために通られた、桁数がめちゃくちゃおおい次世代の住所規格
- リンクローカル：インターネット全体では使えず、同じケーブルにつながっている隣の人とだけ通信できる限定的なモード→マンション内の内線番号
